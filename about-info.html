HyperGrid Chess

A modern chess game featuring an AI opponent powered by Stockfish, built with web technologies.

Features

  • Play against AI with five difficulty levels (Beginner, Club, Advanced, Maximum, Blindfold)
  • Two-player local mode for playing with a friend
  • Multiple time controls (Blitz, Rapid, Classical)
  • Save and load your games
  • Sound effects for moves, captures, and game events

Credits

Chess piece images courtesy of Lichess.

AI engine powered by Stockfish 18 (stockfish.js by nmrugg).

Technologies

HTML5 CSS3 JavaScript Chess.js Chessboard.js Stockfish 18 WebAssembly Web Workers

Stockfish 18 (WASM)

The world's strongest chess engine, compiled to WebAssembly via stockfish.js. Runs entirely in the browser using a Web Worker to avoid blocking the UI. Supports UCI protocol for configuration of skill level, search depth, and move time limits.

Chess.js

A JavaScript chess library for move generation/validation, piece placement/movement, and check/checkmate/stalemate detection. Handles all game rules including en passant, castling, and pawn promotion.

Chessboard.js

A standalone JavaScript chessboard widget that renders an interactive board with drag-and-drop piece movement. Supports board orientation flipping, custom piece themes, and position setup via FEN strings.

WebAssembly & Web Workers

Stockfish runs as a WASM module inside a dedicated Web Worker thread, keeping the main thread responsive. The engine communicates via postMessage, allowing non-blocking AI calculation with configurable depth and time limits per difficulty level.

Canvas Animation

The animated cyberpunk grid background is rendered on an HTML5 Canvas element, providing the signature visual aesthetic of HyperGrid Chess with minimal performance impact.
